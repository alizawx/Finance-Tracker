Finance Tracker

A modern personal finance management system built with Django.

Overview

Finance Tracker helps users manage their personal finances by organizing accounts, tracking income and expenses, and monitoring their financial activities in one place.

This project is being developed as a learning project to practice software design, database modeling, and Django best practices while building a production-style web application.

Features

Current

* User authentication
* Account management
* Income tracking
* Expense tracking
* Transaction history
* Category management

Planned

* Monthly budgets
* Financial dashboard
* Charts and reports
* Search and filtering
* Pagination
* Export to Excel/PDF
* Email notifications
* REST API
* Docker support
* Automated tests

Tech Stack

* Python
* Django
* PostgreSQL
* HTML
* CSS
* JavaScript

Project Goals

* Learn software architecture and database design.
* Apply Django best practices.
* Build a scalable and maintainable web application.
* Create a portfolio-ready project.
